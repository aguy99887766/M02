Binary:

	Converts any binary to a decimal number
	Do not add any spaces with your input
	Only use the numbers 1 or 0
	
	Formula
		the sum of u(2^n) from 0 to n
		where u is equal to a 1 or 0
		n is the current index

Infix/Postfix:

	Removes any parenthesis around an input
	Input:
		(A+B)
	Output:
		A+B

	Do not add any spaces
